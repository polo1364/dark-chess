# 象棋暗棋（翻翻棋）— 單檔版（無連吃）

- 4×8 直式棋盤，手機/桌機皆適配
- 人機 / 雙人；AI 思考提示；勝負 / 和局判定
- 棋子中文字、不半透明；**每回合一動作**（翻 / 移一步 / 吃一枚）
- 首翻決定持色，玩家名稱旁顯示 **紅 / 黑** 徽章
- 提示欄紀錄動作；**認輸** → 結果覆蓋層 → 新局
- AI 使用 snapshot-safe 規則，不與全域狀態混用，避免卡住

## 使用
1) 下載 `dark-chess-release.zip`，解壓後開啟 `index.html` 即可。
2) 若看見舊畫面，網址後加上 `?v=fix` 以清快取。
